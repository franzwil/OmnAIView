Colors for hovering over buttons:
- green for positive (saving)
- red for attention (deleting)
- lighter color for nothing special

### Button states

1. Start (to 2), Record (to record menu and then to 4)
2. Stop (to 3), Record (not available)
3. Start (to 2), Save (to save menu (**green?**) and then to 1), Record (to record menu and then to 4)
4. Start (not available), Stop (to save menu (**green?**) and then to 1), delete (**red**, to 1)
